The Presidents Prints

Tour

Fingerprints are presented with the object upon which they were found. Objects were collected in public spaces and represent everyday items used by each of us, potentially daily.

Each work is a pen plot of a fingerprint vectorisation. Each plot is 30 cm by 40 cm on 300 gram paper. Each plots is assigned an eight digit hexadecimal number that becomes its name. Pen plots were generated using an AxiDraw machine and various different pens.

The representation of a fingerprint as a hexadecimal number is a reminder that digital devices do not store images of the fingerprint. Instead devices store a cryptological representation called hash. A specific fingerprint can generate a specific hash but a specific hash cannot recreate the fingerprint that generated it.

Thus stolen devices do not allow the construction of the fingerprint that allows the device to be used. There is no backward relationship between a hash and the fingerprint that generated the hash.
